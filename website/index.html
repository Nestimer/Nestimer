Frequently Asked Questions

Do I need to install anything on my phone?

No. You manage everything from your browser at my.nestimer.com. Works on any phone, tablet, or computer. There's also an optional native app for iOS and macOS if you prefer.

Is it really free?

Yes. No subscription, no premium tier, no credit card required. All features are included for unlimited devices.

Can my child bypass the limits?

The agent installs as a root-protected system service. It cannot be quit, and if killed, a watchdog restarts it within 15 seconds. If your child is a Standard User (not admin) on macOS, they cannot remove it.

What if the internet goes down?

All rules work offline. The agent enforces limits and downtime locally. You can generate offline unlock codes from the dashboard — the code changes every 5 minutes and gives 30 minutes of free time.

How is this different from Apple Screen Time?

Apple Screen Time is widely known to be easy for children to bypass. NesTimer is root-protected, offers per-day limits, class windows with buffer time, offline unlock codes, and doesn't require iCloud Family Sharing.

What Mac does my child need?

Any Mac running macOS 13 (Ventura) or newer. Both Intel and Apple Silicon (M1/M2/M3/M4) are supported.

Does it track what my child does online?

No. NesTimer only tracks total screen time. It does not monitor browsing, apps, messages, or any content. It's a time management tool, not surveillance.

Can I use it for multiple children?

Yes. Add as many devices as you need, each with its own rules, schedules, and activity windows.

I'm technical. Can I self-host it?

Yes! NesTimer is fully open-source. Clone the GitHub repo, run docker compose up, and you have your own private instance. The cloud dashboard at my.nestimer.com is the same software.
